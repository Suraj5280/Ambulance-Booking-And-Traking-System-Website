# Ambulance-Booking-And-Traking-System-Website
🚑 Ambulance Booking and Tracking System


📌 Project Overview
The Ambulance Booking and Tracking System is a web-based application developed to provide quick and efficient ambulance services during emergencies. The system allows users to book ambulances, track their status, and manage requests in real-time.This project is designed as a BSc Computer Science final year project and aims to improve emergency response time and healthcare service management.


🎯 Objectives
(1)To provide an easy way to book ambulances online
(2)To reduce delay in emergency medical services
(3)To track ambulance status and availability
(4)To manage users, drivers, and bookings efficiently

👨‍💻 Features

👤 User Module
(1)User Registration & Login
(2)Book Ambulance
(3)View Booking History
(4)Track Driver Status

🚑 Driver Module
(1)Driver Login
(2)Update Status (Available / Busy)
(3)Booking info location and pationt details

🛠️ Admin Module
(1)Viwe Users
(2)Manage Drivers
(3)Manage Bookings
(4)Dashboard with system statistics
